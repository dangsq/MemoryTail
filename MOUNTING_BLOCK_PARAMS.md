# 安装座参数调整指南

## 📦 功能说明

在尾巴的第一个链节前面添加了一个正方体安装座，用于卡在机器狗屁股的两半之间。

## 🎛️ 可调参数

在 GUI 的 "Mounting Block" 面板中，你可以调整以下参数：

### 1. **尺寸参数 (Size)**

- **Size X**: 正方体的 X 轴尺寸（宽度）
  - 范围: 0.005 - 0.1
  - 默认值: 0.02
  - 单位: 米

- **Size Y**: 正方体的 Y 轴尺寸（深度）
  - 范围: 0.005 - 0.1
  - 默认值: 0.02
  - 单位: 米

- **Size Z**: 正方体的 Z 轴尺寸（高度）
  - 范围: 0.005 - 0.1
  - 默认值: 0.01
  - 单位: 米

### 2. **位置参数 (Position)**

- **Position X**: 正方体的 X 轴位置（左右）
  - 范围: -0.1 - 0.1
  - 默认值: 0.0
  - 单位: 米

- **Position Y**: 正方体的 Y 轴位置（前后）
  - 范围: -0.1 - 0.1
  - 默认值: 0.0
  - 单位: 米

- **Position Z**: 正方体的 Z 轴位置（上下）
  - 范围: -0.1 - 0.1
  - 默认值: -0.005
  - 单位: 米

### 3. **启用/禁用**

- **Enabled**: 是否显示安装座
  - 默认值: true

## 🔧 调整步骤

1. 打开浏览器访问 http://localhost:5175/MemoryTail/
2. 在右侧 GUI 面板中找到 "Mounting Block" 文件夹
3. 调整参数，实时查看效果
4. 找到合适的参数后，记录下来
5. 将参数告诉我，我会更新到代码中

## 🎨 材质说明

安装座使用灰色金属材质：
- 颜色: 灰色 (#808080)
- 金属度: 0.8
- 粗糙度: 0.2

## 📐 关节角度更新

根据 SCAD 文件分析，球窝关节的最大角度已从 35° 更新为 **45°**。

## 🔍 坐标系说明

- **X 轴**: 左右方向（正方向为右）
- **Y 轴**: 前后方向（正方向为前）
- **Z 轴**: 上下方向（正方向为上）

尾巴的第一个链节位于原点 (0, 0, 0)，安装座在其下方/前方。
